# Bill Splitter

A small Svelte app that calculates a tip and splits a restaurant bill.

## Run it

```bash
npm install
npm run dev
```

Then open the URL it prints (usually http://localhost:5173).

## Your job (QA)

Use the app like a normal person would. Try different bill amounts, tip
percentages, and numbers of people. Watch the numbers at the bottom.

There are a few bugs in here. You don't need to read or touch any code — just
poke at the interface and write down anything that looks wrong, with the steps
to reproduce it. Treat it like a real bug report:

- **What you did** (e.g. "set bill to 50, picked 20% tip")
- **What you expected** (e.g. "tip should be $10")
- **What actually happened** (e.g. "tip showed $1")
